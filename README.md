# Simple Blog System using MERN

This project is a simple blogging system built with React for the frontend and Express for the backend.

## Getting Started

To get the project running, you will need to set up both the client and server parts of the project.

### Prerequisites

- Node.js
- MongoDB Service running

### Installation

First, clone the repository and navigate into it:

```bash
git clone https://github.com/Tyrese-FullStackGenius/MERN_Simple-Blog-System.git
cd simple-blog-system
